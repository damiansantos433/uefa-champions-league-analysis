# UEFA Champions League Data Analysis

A Python data analysis project exploring UEFA Champions League match results from 2004 to 2021. Driven by a passion for soccer, this project digs into team performance, scoring trends, and which countries have historically dominated Europe's biggest club competition.

## About the Project

As a soccer fan, I wanted to go beyond just watching the games and actually understand the data behind them. Using a real dataset of Champions League matches, I analyzed team and country-level performance to answer questions like: Who wins the most? Who scores the most? And which country produces the best Champions League teams?

## Features

- Loads and cleans real UEFA Champions League match data
- Calculates team-level stats including wins, goals scored, goals conceded, and goal difference
- Identifies the top and bottom performing teams across all seasons
- Finds which teams have the best and worst defensive records
- Analyzes the correlation between scoring and winning
- Maps teams to their countries and ranks nations by total wins

## Technologies Used

- Python
- pandas

## How to Run

1. Clone or download the repository
2. Install dependencies:
```
   pip install pandas
```
3. Run the analysis:
```
   python final_project.py
```

## Dataset

The dataset used is `UEFA_Champions_League_2004-2021.csv`, which contains match-level data including home and away teams and scores across all Champions League seasons from 2004 to 2021.
